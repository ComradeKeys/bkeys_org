3D_Snake

3D_Snake began during the summer of 2015, it is a remake of the classical arcade game Nibbles, commonly referred to as Snake. The only exception about my version is the board is tilted, giving it a 3D effect. It is almost complete. It has little features currently, however it should feature customizable controls when I complete it. Some of the utilities I used for this project are:

    Utilities used:

  • GCC C compiler
  • FreeGlut
  • OpenGL

    Languages used:

  • C99

GL_Less

GL_Less is a collection of presentations I created with Dane Christensen for an ACM sanctioned class taught at BYU-Idaho. It's main aim is to take a person completely inexperienced with graphics and to elevate them to an intermediate level of understanding, covering GLSL, shaders, and other subjects surrounding OpenGL. I do not teach the class personally however Dane is teaching them the fall of 2015.

    Utilities used:

  • pdflatex engine
  • evince document viewer
  • OpenGL
  • FreeGlut
  • SDL2

    Languages used:

  • LaTeX
  • Shutter (for screenshots)

    Languages used:
